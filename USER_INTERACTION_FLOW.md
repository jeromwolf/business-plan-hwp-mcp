## 2. 포맷 선택 플로우

### 2.1 정부 지원사업 포맷

```
User → "정부 지원사업 사업계획서"
         ↓
Claude → 지원사업 유형 확인
         ├─ 창업지원 (예비창업, 초기창업)
         ├─ R&D 지원 (기술개발, 연구과제)
         └─ 사업화 지원 (수출, 마케팅)
         ↓
      포맷별 필수 섹션 생성
         ├─ 사업 개요
         ├─ 시장 분석 (Excel 필요)
         ├─ 기술/제품 설명 (이미지 필요)
         ├─ 사업 추진 계획
         ├─ 예산 계획 (Excel 필요)
         └─ 기대 효과
         ↓
      HWP 생성
```

### 2.2 VC 투자 포맷

```
User → "VC 투자용 사업계획서"
         ↓
Claude → 투자 단계 확인
         ├─ Seed
         ├─ Series A
         └─ Series B+
         ↓
      투자자 관점 섹션 생성
         ├─ Executive Summary
         ├─ Problem & Solution
         ├─ Market Size (Excel 필요)
         ├─ Business Model
         ├─ Traction (차트 필요)
         ├─ Team
         └─ Financial Projections (Excel 필요)
         ↓
      HWP 생성
```

### 2.3 은행 대출 포맷

```
User → "은행 대출용 사업계획서"
         ↓
Claude → 대출 목적 확인
         ├─ 운전자금
         ├─ 시설자금
         └─ 창업자금
         ↓
      신용평가 중심 섹션 생성
         ├─ 사업자 정보
         ├─ 사업 현황
         ├─ 재무제표 (Excel 필수)
         ├─ 담보/보증 현황
         ├─ 자금 사용 계획
         └─ 상환 계획
         ↓
      HWP 생성
```

## 4. 데이터 입력 인터페이스

### 4.1 텍스트 정보 입력

```
Claude: "사업 정보를 알려주세요"

User 입력 방식:
1. 자유 텍스트: "우리는 AI 교육 스타트업..."

   - 회사명: OO테크
   - 설립일: 2024.01
   - 대표자: 김OO
3. 문서 참조: "이전 대화 내용 참고해서"
```

## 7. 에러 처리

```
⚠️ 주의 사항:

1. Excel 파일 오류
   "Excel 파일을 읽을 수 없습니다. 
    파일이 손상되었거나 지원하지 않는 형식입니다."
   → 해결: 다른 형식으로 저장 후 재시도

2. 이미지 크기 초과
   "이미지가 너무 큽니다 (최대 10MB)"
   → 해결: 이미지 압축 후 재업로드

3. 필수 정보 누락
   "재무 정보가 없습니다. 
    Excel 파일을 업로드하거나 직접 입력해주세요."
   → 해결: 데이터 제공
```

## 8. 단축 명령어

```
빠른 명령어:

"정부 사업계획서" → 정부 지원사업 포맷
"투자 유치" → VC 포맷
"대출 신청" → 은행 포맷
"재무 추가" → Excel 업로드 요청
"이미지 넣어줘" → 이미지 업로드 요청
"완성해줘" → HWP 생성
"수정" → 특정 섹션 수정
```
