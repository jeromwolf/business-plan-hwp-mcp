## 핵심 GUI 설계 원칙

### 1. 인코딩 처리 최우선
- 모든 단계에서 인코딩 상태 표시
- 실시간 깨짐 감지 및 자동 수정
- 테스트 도구 제공

### 2. 단순하고 직관적인 UI
- 3단계 프로세스: 업로드 → 확인 → 생성
- 드래그&드롭 중심
- 실시간 미리보기

### 3. 에러 방지
- 사전 검증
- 자동 수정 제안
- 되돌리기 기능

### 4. Mac 최적화
- 네이티브 Mac UI 스타일
- 단축키 지원
- Retina 디스플레이 대응
